# Lab-3-ML

# ARTI308 – Lab 3: Exploratory Data Analysis (EDA)



---

## Project Overview
This project applies Exploratory Data Analysis (EDA) techniques to a Movies dataset.  
The goal is to explore, understand, and analyze the dataset using statistical summaries and visualizations.

---

## Dataset
The dataset used in this project is:

**Movies Dataset (movies.csv)**

The dataset contains information about movies, including (depending on the columns):
- Movie title
- Release date
- Runtime
- Budget and revenue
- Vote average and vote count
- Original language
- Genres, keywords, cast, director (text-based fields)

---

## EDA Techniques Applied
The following EDA steps were performed:

- Data inspection using:
  - `head()`
  - `info()`
  - `describe()`
- Checking for missing values
- Data cleaning (handling null values and invalid values)
- Data visualization (histograms, boxplots, countplots, scatterplots)
- Correlation analysis
- Observations and conclusions

---

## Files Included
- `movies.csv` → Dataset file  
- `lab3movies.ipynb` → Jupyter Notebook containing the EDA analysis  

---

## Conclusion
Through this analysis, patterns and relationships between variables were identified (e.g., how budget relates to revenue, trends over release years, and how ratings distribute across the dataset).

This project demonstrates the application of EDA techniques to understand real-world datasets before applying machine learning models.
